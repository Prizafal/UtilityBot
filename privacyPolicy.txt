# Privacy Policy for UtilityBot

## Data Collected by UtilityBot

UtilityBot is open sourced, self-hosted Discord bot software, designed for server administration, moderation, and community management. The bot only processes information necessary to perform commands and features requested by authorized server administrators.

Depending on how UtilityBot is configured and used, the bot may process the following information:

### Discord Identifiers

* Discord User IDs
* Discord Server (Guild) IDs
* Discord Role IDs
* Discord Channel IDs
* Discord Message IDs

### Role and Membership Information

UtilityBot includes features that allow server administrators and authorized moderators to:

* View which users possess specific roles
* Export lists of users associated with a role
* Export Discord User IDs associated with role membership
* Add roles to users individually or in bulk
* Remove roles from users individually or in bulk
* Generate CSV files containing Discord User IDs and role information

### Message and Activity Information

UtilityBot may access and process message-related information when required to perform moderation and administrative functions, including:

* Message content
* Message timestamps
* Message author User IDs
* Channel and server information associated with messages

UtilityBot may use this information to:

* Determine whether a user has sent messages during a specified time period
* Identify inactive members
* Generate activity-related reports
* Enforce moderation policies
* Delete messages that meet administrator-defined criteria
* Execute moderation and server management commands

UtilityBot performs these actions only when requested through a command or feature initiated by an authorized administrator or moderator. UtilityBot does not continuously monitor, profile, or track user activity beyond what is necessary to execute requested commands.

### Temporary Processing

Information may be temporarily processed in memory while commands are executed. This may include role information, message information, user identifiers, and command parameters required to generate reports or perform moderation actions.

Unless separately configured by the host, UtilityBot does not retain message content, activity data, or user information beyond what is necessary to complete the requested operation.

### Generated Reports and Exports

UtilityBot may generate reports, CSV files, or other exports containing information obtained from Discord, including:

* Discord User IDs
* Discord usernames and display names
* Role membership information
* Activity information derived from message history and timestamps
* Moderation-related information

These reports are generated only in response to actions taken by authorized server administrators or moderators.

Generated reports may be downloaded, stored, or shared by the administrator operating the bot instance. UtilityBot does not control how administrators use exported data after it has been generated.

Because UtilityBot is self-hosted, any generated reports are stored and controlled by the operator of the specific bot instance rather than by the developer of UtilityBot.

No information processed by independently hosted instances of UtilityBot is automatically transmitted to, collected by, or made accessible to the original developer of UtilityBot (Prizafal).

## Self-Hosted Use Only

UtilityBot is distributed exclusively as self-hosted software. The developer does not operate a public UtilityBot service.

Individuals and organizations who deploy UtilityBot are solely responsible for the data processed by their instance of the software.

The developer of UtilityBot does not automatically collect, receive, store, or have access to information processed by independently hosted instances, including:

* Discord User IDs
* Role membership exports
* Generated CSV files
* Moderation logs
* Activity reports
* Message content processed by moderation features
* Records of deleted messages
* Server configuration data

Any information processed by UtilityBot remains under the control of the administrator operating that specific bot instance unless the host separately configures logging, backups, analytics, or external integrations.

## Administrator Responsibility

Server administrators are responsible for informing users about the presence and use of UtilityBot within their servers.

By deploying UtilityBot, administrators acknowledge that the software may be capable of:

* Collecting Discord User IDs associated with server roles
* Generating exports of role membership information
* Generating reports containing Discord User IDs and related server information
* Determining whether users have been active during specified periods
* Processing message content and metadata when required by moderation features
* Deleting messages based on administrator actions or configured moderation rules
* Temporarily processing Discord User IDs and related metadata as necessary to execute commands and generate reports

Administrators are responsible for ensuring their use of UtilityBot complies with applicable laws, Discord's Terms of Service, and any privacy obligations applicable to their communities.

## Bot Identity and Naming

UtilityBot is open-source software that may be deployed under names, usernames, avatars, branding, or server-specific identifiers chosen by the administrator operating a particular instance.

References to "UtilityBot" within this Privacy Policy refer to the software itself and any substantially unmodified or modified self-hosted instance of the software, regardless of the name, branding, profile image, username, or other identifying characteristics used by the operator.

The capabilities and data-processing practices described in this Privacy Policy apply to the software's functionality and are not dependent on the name under which a particular instance is deployed.

The operator of a self-hosted instance may modify the software, add additional features, integrate third-party services, or alter data-processing behavior. In such cases, the operator is responsible for providing any additional privacy disclosures required by those modifications.
